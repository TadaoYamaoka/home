TR-8 TB-3同期再生ソフト TR8TB3Sync

RolandのTR-8 と TB-3のミキシングとMIDI同期を行うソフトです。

USBケーブルで接続したTR-8とTB-3を同期させて再生できます。TR-8のスタート/ストップ、テンポに連動します。

TR-8、TB-3を再生デバイスとしてしても使用できます。TR-8を再生デバイスとした場合は、TR-8のアナログ出力にUSBケーブル経由で入力されたTB-3の音声がミックスされます。

TR-8、TB-3のどちらかのみを接続した状態でも再生可能です。

再生デバイスをTR-8、TB-3以外のオーディオインターフェイスにする場合は、96kHz、32bitもしくは24bitに対応している必要があります。

WASAPI排他モードを使用しているため、高音質、低遅延での再生が可能です。

動作環境

  • Windows 7以上

ダウンロード

GitHubからダウンロード

ソースコード

GitHubでソースコードを公開しています。改良に協力してくださる方を募集します。

使い方

Output Deviceから再生に使用するデバイスを選択して、[Ready]ボタンをクリックします。TR-8もしくはTB-3のスタートボタンを押すと再生が開始します。

クリッピング処理を行っていないのでボリュームを絞った状態から開始してください。ボリュームを上げ過ぎると音割れが発生します。
